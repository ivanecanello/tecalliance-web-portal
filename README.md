K# WebPortal Tecalliance - Programming Practice

A modern web application for managing personal tasks, built with 
**Angular 19** using the latest platform features such as Signals, 
Computed properties, and reactive programming.

## Features

### Main Functionalities
- **Email-based Authentication**: User validation against JSONPlaceholder API
- **Full CRUD Tasks**:
- **Reactive State Management**: Using Angular 19 Signals
- **Route Protection**: Guards for authenticated routes
- **Intuitive Interface**: Material Design + Tailwind CSS

### Technical Features
- **Standalone Components**: Modern architecture without modules
- **Signals & Computed**: Reactive state management
- **Strict TypeScript**: Type-safe throughout the application
- **API Integration**: JSONPlaceholder API consumption
- **Material Design**: Professional components
- **Tailwind CSS**: Responsive and customizable styles

## Requirements

- **Node.js**: v18 or higher
- **npm**: v9 or higher
- **Angular CLI**: v19

## Installation

### 1. Clone the repository
```bash
git clone https://github.com/ivanecanello/tecalliance-webportal.git
cd tecalliance-webportal
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the application
```bash
npm start
```

The application will be available at `http://localhost:4200`
